# Definition of Done (DoD)

A PBI or task is considered "Done" only when:
- [ ] Code is reviewed and approved by at least one other team member.
- [ ] Acceptance criteria are met and verified.
- [ ] Documentation is updated (CHANGELOG.md, README.md, etc.).
- [ ] All issue-linked PRs are merged.
- [ ] Automated tests pass.
- [ ] No PII or sensitive credentials are committed.
